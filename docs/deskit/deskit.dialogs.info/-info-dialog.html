InfoDialog

@Composable
fun InfoDialog(width: Dp = 450.dp, height: Dp = 230.dp, resizable: Boolean = false, title: String = "Information", message: String = "Information message", icon: Painter? = null, colors: InfoDialogColors = InfoDialogDefaults.colors(), iconSize: DpSize = DpSize(64.dp, 64.dp), onClose: () -> Unit, content: @Composable () -> Unit = { Text( text = message, style = MaterialTheme.typography.bodyLarge, textAlign = TextAlign.Center, modifier = Modifier.fillMaxWidth() ) })

Displays a modal information dialog.

This dialog is designed to present information that requires user acknowledgment. It features a title, a customizable content area, an optional icon, and a single "OK" button to close it.

Parameters

width

The initial width of the dialog window.

height

The initial height of the dialog window.

resizable

Whether the user can resize the dialog window. Defaults to false.

title

The text displayed in the dialog window's title bar.

message

The default message text. This is used by the default content lambda. If you provide a custom content composable, this parameter is ignored.

icon

An optional Painter to be displayed at the top of the dialog, above the content.

colors

Default colors for Info dialog.

iconSize

The size of the icon.

onClose

A callback function that is invoked when the user clicks the "OK" button or closes the dialog window.

content

The main content of the dialog. By default, it displays the message text. This can be overridden with any custom Composable content for more complex layouts.
